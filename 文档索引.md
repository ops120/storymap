```
开始使用.md → 安装指南.md → LLM配置示例.md → 快速参考.md
```

学习目标：
- ✅ 能够独立安装和启动服务

```
功能说明.md → 项目结构.md → CHANGELOG.md
```

```
众生谱项目需求与架构文档.md → 项目结构.md → 源代码
```

学习目标：
- ✅ 理解系统架构和设计思路
- ✅ 能够修改和扩展功能

- ✅ 能够贡献代码
