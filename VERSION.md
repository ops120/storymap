## 升级指南

### 从 v2.3 升级到 v2.4

1. 拉取最新代码：
   ```bash
   git pull origin main
   ```

2. 重启后端服务（修复了 SQL 错误）：
   ```bash
   # 停止旧服务，然后
   start_backend.cmd
   ```

3. 刷新前端页面

4. 建议运行"清理重复"功能清理数据库中的孤立边

### 从 v1.0 升级到 v2.4

1. 备份数据库文件 `storymap.db`
2. 拉取最新代码
3. 重新安装依赖（如有更新）
4. 重启服务
5. 测试功能是否正常
